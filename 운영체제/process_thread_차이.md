```
프로세스는 운영체제로부터 자원을 할당받는 작업의 단위이다
스레드는 할당 받은 자원을 이용하는 실행의 단위이고 프로세스 내에 여러개 생길 수 있다
어플리케이션 하나가 프로세스이고, 그 안에서의 분기 처리가 스레드가 되는 셈이다
```

```
process 에는 code, data, stack, heap 영역으로 구성된다. 
thread는 메모리를 서로 공유한다. 
process가 할당받은 메모리 영역 내에서 stack 형식으로 메모리 영역은 따로 할당받고,
나머지 code/data/heap 형식으로 할당된 메모리 영역은 서로 공유한다.
-> 각각의 thread는 별도의 stack을 가지고 있고 heap 메모리 영역은 서로 공유한다. 

스레드는 Code/Data/Heap 메모리 영역의 내용을 공유하기 때문에 어떤 스레드 하나에서 오류가 발생한다면 같은 프로세스 내의 다른 스레드 모두가 강제로 종료된다.
```
