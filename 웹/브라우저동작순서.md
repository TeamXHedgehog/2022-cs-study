URL 요청을 하면 DNS에 브라우저가 요청을 해서
IP를 찾아 전송시킨다.

클라이언트와 서버는 3way Handshake를 통하고 데이터 전송이 시작되고
HTTP Request, HTTP Response 를 주고 받는다.

그 뒤 사용자에게 받은 데이터를 브라우저로 보여줄 때 W3C 명세에 따라
HTML과 CSS를 파싱하여 DOM 트리와 CSSOM 트리를 만든다.

그러다가 자바스크립트 태그를 만나면 JS Engine을 돌려 자바스크립트 엔진을 해석하여(인터프리터 방식임)
추상 구문 Tree인 AST를 만들고 실행한다.

JS 코드 파싱이 완료가 되면 다시 html 파싱 작업을 하여 완료 시킨다.

그 뒤 DOM과 CSSOM 트리가 합쳐져 Render Tree를 만들고 이 렌더링 트리에서 노드들을 배치하고
 UI를 그려 사용자에게 결과 화면을 출력시킨다.
